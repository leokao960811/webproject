# HW3 專題想法

影片: https://youtu.be/k_U04l7qhqY

針對期末專題，這是我決定使用的3個範例:

1. ToDo App:
  這個範例建立一個待備清單，用於確立要完成的事情。
2. Event Countdown:
  這是個建立活動並顯示剩餘時間的範本，可以用於倒數計時。 
3. Quiz App:
  這個範例是做一個問答的小遊戲，可用來當作教學輔助，如背單字等等。
   
這三個專案會在我們的期末專題中這樣運用：

我們預計建立一個找歌詞、並建立單字庫的專案。
使用者可在歌詞中尋找單字意思並儲存成一個清單，
再以測驗的形式進行問答，強化學習效果。
我們同時預計做一個倒數鐘的小程式，強化學習專心度。
